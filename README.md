# ✨ Martha Cute Studio - The Cute App

Sistema de gestión y reservas profesional para salón de belleza, diseñado como una **PWA (Progressive Web App)** de alto rendimiento.

## 🚀 Tecnologías
* **React + Vite** (Velocidad pura)
* **Tailwind CSS** (Diseño de lujo tipo Bolt)
* **Supabase** (Base de datos y autenticación en tiempo real)
* **Lucide Icons** (Iconografía moderna)
* **Framer Motion** (Animaciones suaves)

## 🛠️ Configuración
1. Clona el repositorio.
2. Instala las dependencias: `npm install`
3. Crea un archivo `.env` con tus credenciales de Supabase:
   ```env
   VITE_SUPABASE_URL=tu_url
   VITE_SUPABASE_ANON_KEY=tu_key
